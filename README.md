## 概要
生体データをgoogle fit/ヘルスケアアプリから取得し、一律で使用できるようにします。

## 構成
fittrack_server：サーバ用のコード（RailsをAPIモードで使用）
fittrack_ui：フロントエンド用コード（Flutterを使用）

## バージョン
Ruby on Rails：6.0.3.4
Flutter：1.27.0-2.0.pre.101（2021/02/15時点）
